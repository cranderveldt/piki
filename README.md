Blue Select
===========

This is kkinnable, Blue State-created select replacement jQuery plugin.

## Usage

```javascript
jQuery(document).ready(function($){
    $('select').blueSelect();
});
```

## Options

Right now the only option is `icon` which you use like this:

```javascript
jQuery(document).ready(function($){
    $('select').blueSelect({
        'icon' : '&caron;'
    });
});
```
